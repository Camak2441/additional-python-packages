# Additional Python Packages

To use this flake, add the url "github:Camak2441/additional-python-packages" as an input to a flake. 

## What is this?

This is a flake which holds whichever python packages I needed, but weren't already packaged on nixpkgs. The packaging here is jank, and is not assured to work,and is only compatible with python 3.13. 

## What is Packaged in this?

The following python packages are packaged in this flake:

```
drjit
mitsuba
scikit-sparse
```
